Programacion Orientada a Objetos

En este curso se te enseñara como se creará un programa que esté totalmente orientado a objetos, aplicando herencia y polimorfismo donde se simulará un juego de cartas mayor entre dos jugadores.


Se empieza el código declarando que se creara una baraja inglesa y luego se crea poniendo dígitos y características de las cartas aleatoriamente.



Se establecen variables iniciales, es decir, se crea la clasificación de los jugadores y las rondas como por ejemplo en este se crea los puntos que tiene este personaje ficticio llamado “Rafa” y “Héctor” y en la siguiente columna se crea el número de rondas que han transcurrido durante el juego.



Se crea el juego, se hace un código donde se reparten las cartas al azar para cada jugador y a cada jugador le toca una carta distinta.



Se comparan cartas y se suman los puntos, es decir el jugador que haya juntado más puntos por obvias razones gana el juego.



Se imprime quien fue el ganador y con cuántos puntos gano dependiendo de la persona que haya acumulado más, y si ambos jugadores tienen la misma cantidad de puntos se imprime un “Hubo un empate”.
